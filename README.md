# Цифровой фонокорректор

## Назначение
Утилита командной строки для цифровой коррекции аудиосигнала по кривой RIAA.

## Требования к входному файлу
*   **Формат:** WAV (несжатый).
*   **Каналы:** Моно или Стерео.
*   **Частота дискретизации:**
    *   **192 кГц / 24 бит** — рекомендуется (лучшая линейность АЧХ).
    *   **96 кГц** — допустимо.
    *   **48 кГц** — не рекомендуется (спад АЧХ -10 дБ на 20 кГц).

## Параметры командной строки
```text
RIAA_wave_convert.exe <файл> [Коэффициент] [Путь]
```

| Параметр | Статус | По умолчанию | Описание |
| :--- | :---: | :---: | :--- |
| **Имя файла** | Обязательный | — | Исходный WAV файл. |
| **Коэффициент (Ку)** | Необязательный | 16 | Уровень усиления. Ку=1 соответствует -20 дБ на 1 кГц. Подбирается для избежания клипинга. |
| **Путь** | Необязательный | Папка программы | Директория исходного файла (если отличается). |

## Примеры использования
```bash
RIAA_wave_convert.exe mytrack.wav 21.5 "D:/Test tracks/project01"
RIAA_wave_convert.exe mytrack.wav 18.3
RIAA_wave_convert.exe mytrack.wav
```

## Результат работы
*   **Расположение:** Та же папка, что у исходного файла.
*   **Формат:** Аналогичен входному файлу.
*   **Обработка:** Идеальная кривая RIAA (без сабсоник-фильтра).
